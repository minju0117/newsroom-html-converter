# Newsroom HTML Converter

GitHub Pages에서 무료로 띄워 쓰는 뉴스룸 관리자 HTML 변환기입니다. 파일은 서버로 업로드되지 않고 브라우저 내부에서만 처리됩니다.

## 지원 파일

- `.docx`: 보도자료형 원고
- `.eml`: 이메일 원문
- `.pptx`: PPT 뉴스룸 초안
- `.html`, `.txt`: 기존 코드 또는 일반 텍스트

## 사용 방법

1. GitHub Pages 주소에 접속합니다.
2. PPTX, DOCX, EML 중 하나를 업로드합니다.
3. 필요하면 제목을 직접 수정합니다.
4. 홈페이지 관리자에서 이미지를 업로드한 뒤 생성된 `/upload/editor/...` URL을 `이미지 URL`에 한 줄씩 입력합니다.
5. 영상이 있으면 YouTube URL을 입력합니다.
6. `변환하기`를 누릅니다.
7. 미리보기를 확인하고 `HTML 복사` 또는 `HTML 다운로드`를 사용합니다.

## 샘플 기준 반영 규칙

### DOCX 보도자료형

- `제목:` 줄은 제목 메타로 사용하고 본문에서는 제거
- 제목은 `<p class="tit_mid center bold">`
- `-`로 시작하는 요약 bullet은 `<p class="bold">`로 묶음
- `■사진자료:`는 이미지 하단 캡션인 `<p class="center img_below_txt">`로 이동
- 나머지 문단은 일반 `<p>`와 `<p><br></p>` 간격으로 출력

### EML 메일형

- 메일 앞부분 인사말, 업로드 요청 문구, 서명, 이메일 고지문 제거
- `제목:` 이후 본문 중심으로 변환
- 첨부/inline 이미지는 자동 URL을 알 수 없으므로 관리자 업로드 후 URL 입력
- `contact us` 등 링크 문구는 원문 추출 후 수동 검수 권장

### PPTX 뉴스룸형

- 슬라이드 텍스트 박스에서 텍스트 추출
- `뉴스룸`, `이미지 1`, `영상 1` 같은 작업용 라벨 제거
- 짧은 섹션 제목은 네이비 bold 스타일 적용
- 이미지/영상 URL은 입력칸으로 반영

## 주의

Public GitHub Pages에 올려도 사용자가 업로드하는 파일은 GitHub로 저장되지 않습니다. 다만 실제 고객사 원본, 비공개 HTML 샘플, 계정 정보는 저장소에 올리지 마세요.
